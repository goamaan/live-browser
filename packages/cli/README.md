# live-browser

CLI package for `live-browser`, including the installable `live-browser` skill payload.

## Install

```bash
bun i -g live-browser
npm i -g live-browser
```

The installed binary is `live-browser`.

## Local development

This package is developed in a Bun-first workspace, but the published CLI remains Node-compatible for end users.

See the root project README for setup, examples, and live-browser usage guidance.
